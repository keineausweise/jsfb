Сделать структуру данных, которая отображает расписание школьника.

1. У школьника 5 учебных дней в неделю, следовательно структура это масив из пяти элементов 
[{...}, {...}, {...}, {...}, {...}]

2. Каждый день должен иметь в себе название (понедельник, вторник, и т.д.). 
Также он должен содежать в себе список уроков в этот день
Т.е. День - это обьект со свойством "name", и массивом "lessons"
{
    name: "Monday",
    lessons: [
        ...
    ]
}

3. Каждый урок должен содержать название урока, оценку за урок и домашнее задание




Уроков в день 4. 

Скрипт должен содержаться прямо в странице, пока обойдемся без подключенных файлов.
При загрузке страницы никаких ошыбок не должно быть.
